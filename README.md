## Project Overview

This repository contains my **Digital Literacy Portfolio Project** created as part of the **CSE0001 – Digital Literacy** course at VIT Bhopal University.

The aim of this project is to develop digital awareness, build a professional online presence, and understand safe and responsible use of digital tools and platforms.

The project is divided into **five tasks**, each corresponding to a module of the course.
